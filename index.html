Sobre o Projeto

Esta calculadora foi desenvolvida como parte do meu portfólio de projetos com HTML, CSS e JavaScript. Ela permite realizar operações matemáticas básicas como adição, subtração, multiplicação e divisão.


O objetivo principal deste projeto é demonstrar habilidades em estruturação de layout (HTML), estilização visual (CSS) e lógica de programação com JavaScript puro, sem bibliotecas externas.


O design foi pensado para ser simples, responsivo e funcional em diferentes dispositivos.
